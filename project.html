Puzzle Master

Puzzle Master is a dynamic learning environment that prioritizes user experience. Users have the flexibility to select from various difficulty levels for puzzles and challenges.
In "Puzzle Master," we prioritized security with the following measures:
1. Password Security:
- Implemented bcrypt for secure password hashing, safeguarding user credentials.
2. Input Validation:
- Employed input validation to ensure accurate user details, enhancing data integrity.
3. Comprehensive Testing:
- Conducted rigorous manual and automated testing to identify and address vulnerabilities.
4. JWT Authentication:
- Used JSON Web Tokens for secure user authentication, ensuring efficient data transmission.
5. Robust Session Management:
- Implemented strong session techniques with secure cookies to prevent unauthorized access.
6. Limited Session Duration:
- Configured session cookies with a 30-minute maximum duration for added security.
